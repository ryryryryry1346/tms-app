# Legacy Flask App

This directory contains the pre-migration Flask/Jinja version of TMS.

It is kept for:

- behavior comparison during the TypeScript migration
- rollback/reference if a legacy flow needs to be inspected
- historical context for routes, templates, and SQL access patterns

It is not the active application that should be deployed going forward.

For ongoing work, use the TypeScript app in [`web/`](C:/myapp/web).
